# My Dotfiles Configuration

## SetUp
Install the following:
- Hammerspoon and Karabiner Elements for key mappings
- Zsh and ohmyzsh
- Iterm2 and Tmux
- Vim
- Z 
- FZF
- fd

```
$ mkdir .dotfiles
$ cd .dotfiles
$ git clone https://github.com/a180024/dotfiles
$ find ~+ -type f -name ".*" ln -sv {} ~ \;
```
